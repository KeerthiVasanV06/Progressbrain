# Progress Brain

A productivity and study tracking app with a React frontend and Node/Express/MongoDB backend.

## Structure
- `client/` - React frontend
- `server/` - Node + Express backend

## Setup
1. Install dependencies in `server/` and `client/`.
2. Configure `.env` for MongoDB.
3. Run backend: `npm start` in `server/`.
4. Run frontend: `npm start` in `client/` (after React setup).
